# Get company project name
Get the name and the id of each projects

**Url:** ```/admin/api/company/get-projects```

**Method:** GET

**Authentification** True

**Permission** ROLE_ADMIN

## Response

```json
[
    {
        "name": "Devaux",
        "id": 123
    }, {
        "name": "Millesimo",
        "id": 145
    }, {
        "name": "Marin",
        "id": 654
    }
]
```
